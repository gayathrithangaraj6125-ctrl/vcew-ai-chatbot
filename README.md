
# VCEW AI Chatbot

This project is an AI-powered chatbot for Vivekanandha College of Engineering for Women.

## Features
- Answers questions about admissions
- Provides information about courses
- Explains departments and placements
- Uses website knowledge for responses

## Technologies Used
- Dify
- Groq Llama 3 model
- Firecrawl
- Retrieval Augmented Generation (RAG)
